API Documentation

Welcome to the ENVOYou API documentation. Here you will find everything you need to integrate with our service.

Getting Started

To start using the ENVOYou API, you will need an API Key. You can get one for free by signing up on the Get API Key page.

Authentication

All requests to the API must be authenticated. Include your API Key in the `Authorization` header with the `Bearer` scheme.

Authorization: Bearer <YOUR_API_KEY>

Endpoints

Get Global Emissions

Fetches verified emissions data for a specific country.

GET /global/emissions

Query Parameters:

  • `country` (string, required) - The ISO 3166-1 alpha-3 country code (e.g., `USA`, `IDN`).

Example Request:

curl -X GET \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  "https://api.veridata.dev/global/emissions?country=USA"

Example Success Response (200 OK):

{
  "country": "USA",
  "year": 2024,
  "co2_emissions_tons": 4870000000,
  "cevs_score": 75.4,
  "source": "EPA",
  "last_updated": "2025-08-24T12:00:00Z"
}
